# 业务映射：AIGC3.0_RoCE流路径交互原型

## 1. 本原型对应的任务类型

- **输出归类**：正式交付物 / 交互原型
- **所属主线**：AIGC / AI Job / RoCE 流路径专项
- **输出位置**：`OutPut/`

## 2. 用户旅程到页面结构映射

根据附件中的手写用户旅程，本原型按如下路径组织：

1. **RoCE业务分析 → AI任务分析**
   - 页面顶部一级导航 + 二级菜单 + AI任务页面主体
2. **AI任务配置创建**
   - 查询表单：开始时间、结束时间、租户、源IP、源端口、目的IP
3. **主要查询条件**
   - 查询区支持时间区间、租户与关键五元组过滤
4. **租户查询 / 全量租户列表**
   - 通过租户下拉实现
5. **流路径列表展示**
   - 列表区展示源/目的、吞吐、时延、抖动、丢包、状态、最近活跃时间
6. **状态 / 操作**
   - 提供“查看详情”“一键诊断”操作
7. **诊断分析 → 故障一键诊断**
   - 通过顶部菜单或页面按钮跳转到独立诊断页
8. **新增检测任务**
   - 在诊断页通过弹层创建任务，诊断类型支持训练中断 / 性能下降 / 训练卡死

## 3. SRS能力到原型模块映射

| SRS / 需求能力 | 原型模块 | 说明 |
|---|---|---|
| 基于五元组查询流路径 | 查询过滤区 | 支持源IP、源端口、目的IP |
| 基于时间范围查询流路径 | 时间选择 + 快捷筛选 | 提供开始/结束时间与最近1h / 4h / 12h / 24h |
| 7天历史留存 | 查询区说明文案 | 通过提示标签说明能力边界 |
| 流路径列表展示 | 主列表 | 展示吞吐、时延、抖动、丢包、状态 |
| 查看路径详情 | 详情抽屉 | 点击单条流进入暗色详情区 |
| 展示设备 / 服务器 / 端口 | 拓扑视图 | 通过 SVG 画布展示节点与接口 |
| 展示吞吐 / 时延 / 丢包 / PFC / ECN 趋势 | 指标趋势页签 | 通过内置轻量 SVG 图示展示 |
| 查看告警信息 | 告警记录页签 | 展示等级、对象、摘要、建议动作 |
| 一级/二级菜单导航 | 顶部导航 + 下拉菜单 | 仅开放“AI任务”“故障一键诊断”两个可点击路径 |
| 发起一键诊断 | 独立诊断页面 | 从列表或详情页都可跳转发起 |
| 新增检测任务 | 任务新增弹层 | 支持训练中断、性能下降、训练卡死 |
