# AIGC3.0_RoCE流路径交互原型

## 1. 交付说明

本目录提供一个可本地直接运行的 `AIGC3.0_RoCE流路径交互原型`，重点覆盖以下评审场景：

- 基于时间范围、五元组、集合通信ID、租户的流路径查询
- 基于自定义起止时间、租户与关键五元组的流路径查询
- RoCE流路径列表展示与状态识别
- 从暗色主界面进入流量详情页
- 拓扑、关键指标趋势、告警记录展示
- 从当前流路径发起“一键故障诊断”

## 4. 原型覆盖范围

### 已覆盖
- 查询过滤区
- 自定义开始/结束时间选择，最长 7 天
- 最近 1h / 4h / 12h / 24h 快捷时间
- 流路径列表
- 流状态与严重等级标识
- 流路径详情抽屉
- 暗色拓扑展示
- 指标趋势图（吞吐 / 时延 / 抖动 / 丢包）
- 告警记录列表
- 一键故障诊断入口与诊断结果卡片

### 暂未扩展
- 真实后端接口联调
- 真正的 G6 / ECharts 图库接入
- 权限模型与租户鉴权

- 真实的诊断算法执行链路

## 5. 设计取舍说明

1. **技术选型**：使用 `HTML + CSS + JavaScript`，避免引入 Node 构建链，适合当前仓库以文档和轻量脚本为主的使用方式。
2. **交互组织**：主页面采用“查询 + 列表 + 详情抽屉”的评审友好模式，既能演示主旅程，也能容纳高保真暗色详情风格。
3. **视觉风格**：主界面与详情页统一为暗色诊断工作台风格，对齐评审版收敛方向。
4. **数据方式**：采用本地 mock 数据，便于后续直接替换为真实接口。

## 6. 建议演示路径

1. 进入页面，保持默认 `最近24小时`
2. 输入或选择 `开始时间 / 结束时间 / 租户 / 源IP / 目的IP`
3. 点击“查询”查看流路径列表
4. 选择异常流，点击“查看详情”
5. 在详情页依次切换“概览与拓扑 / 指标趋势 / 告警记录”
6. 点击“一键故障诊断”查看诊断建议
